03The harness

The harness is what makes the engineer trustworthy.

Behind every skill is a tool that a human domain expert built, proved against known answers, and signed. The model chooses the right tool and explains the result, and it never computes the number itself. Each skill is re-tested whenever a model or a tool changes, so the engineer is verified continuously rather than trusted once.

A

Built by experts

A human domain expert writes and owns each skill, and signs it before the engineer may use it.

B

Proved against truth

Every skill reproduces a set of known answers within tolerance, and is re-examined when anything changes.

C

Traceable by design

Each result carries its inputs, the tool and version that produced it, and the basis it stands on.
